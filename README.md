# Protractor Framework

Required
* nodejs
* npm
* java JRE (java command added to the path)


## Installation
1) Install node dependencies
```
npm install
```

2) Install webdrivers
```
npm run webdrvr-upd
```


## Run tests
Set the variables in:
e2e/resources/test-data-environment.js

```
npm run e2e
```
